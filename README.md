
# 📚 Library Management System Clients

A RESTful API built with **Express.js**, **TypeScript**, and **MongoDB** for managing books in a library system.

---

## 🚀 Features

- 📖 Create, Read, Update, Delete (CRUD) books
- 🧾 Borrow book and track quantity

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- TypeScript
- MongoDB & Mongoose
- dotenv
